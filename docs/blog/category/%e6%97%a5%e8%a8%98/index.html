auからIIJmioにmnp転入した

2016年12月16日 14時43分

auの携帯を使っていたけど2年縛りが終わる今月、料金割引も色々と終了して前月よりも1,500円ぐらい料金が上がって9,000円近くになった。

月の通信容量なんて2Gも超えない程度なので流石にこの値段を払うほど価値を見出せず、mnvoに切り替えることにした。

安いSIMは数多くあったけどある程度は安心感が欲しかったので老舗のIIJmioにした。

手順

  1. IIJmioで使える携帯を取得 -> SIMフリー版 iPhone7購入済み
  2. auからmnp予約番号取得（引き止められるけど世間話して断る）
  3. amazonでSIM申込パックを400円で購入
  4. IIJのホームページから申込
  5. SIMが届いたら付属の用紙に書かれている電話番号に電話しmnp転入手続きを完了させる
  6. 2時間ぐらい待つ
  7. IIJから処理完了メールが届いたら新しいSIMを刺した携帯を再起動して電波が受け取れることが確認できたら完了

ここの情報で再起動することが分かり、処理完了メールがきてもずっと圏外で悩んでいたのでとても助かった。

素晴らしい情報、ありがとうございます。

俺はフリーランスでなくフリーターエンジニアなのかもしれない

2016年12月06日 00時13分

この記事はADVENTAR フリーランス Advent Calendar 2016の6日目の記事です。

前日の記事はさらえみさんのフリーランスイラストレーターを1年以上やってみた。良くも悪くも自分次第。でした。
アニメーター、噂にたがわずとても大変そうな職種のようです…

さて、私のことです。

今年の3月から会社員を辞めてフリーのプログラマとしてお金を稼いで生計を立てています。

猫会社を退職して猫フリーランスエンジニアに転職です。

ただ所謂フリーランスという働き方かというとそうでもないような気がしています。
フリーランスというと案件ベースで請求が発生する方々なイメージで、僕は時間給なのです。
2箇所で働いていますが2箇所とも時間給です。

ん？もしかしてアルバイトというやつで分類としてはフリーターってやつなのか？
でも請求書ベースでやってるからフリーランスっぽい気もする、、、

まぁ細かいことは気にせず、それでやっていけてるのかというととても気楽に稼げています。
週5の8時間労働でも会社員時代よりかなり稼げています。

仕事内容も会社員時代と比べると負担が減り、気楽になっています。
考えられる理由は色々とありますが主に下のような理由が大きいのではないかと思います。

  • 時間給で週2～3日なのでタイトなスケジュールの仕事を与えにくい
  • 正社員でないので責任の大きな仕事を任せにくい
  • 勤務先特有のドメイン知識に長けている
  • どちらかというと正社員様のフォローという役回りになることが多い
  • タイトなスケジュールで仕事を振られても働いた分だけ時給が発生するので納得して働けてる

特に最後の納得して働けてるというのは大きいです。

稀に大きな仕事を担当することがありますが、その際は土日や深夜も作業してスケジュールを間に合わせます。
その際、時給なので働いた分だけ全額支給。金銭面では疑問の入る余地がなく納得しやすいです。

社員にある有給などの安定感は全くありませんが、休んだら休んだだけ所得が減るのは当たり前だと思えるので僕にはこの働き方が合っているように思っています。

このような働き方がいつまでも続くような感じはしませんが、続かなくなれば働き方を変えればいいだけなので気楽に構えています。

今勤めている会社が高負荷・長時間労働・安月給で安定しているのなら働き方を考えても良いのではないでしょうか？
タイムカードの勤務時間から時給計算しているようであればそれは危険信号ですよ。

せっかくのAdvent Calendarなので質問への回答。

1.フリーランスで大事だと思うことは？
人付き合い。
技術がどれだけあっても使いにくい人は結局使ってもらえない。
2.将来どうしたいですか？（法人化、もっと稼ぎたい、もっと時間欲しい）
ここだ！と思える会社に巡り合いたい
3.これからフリーランスを目指す人に一言
フリーランスを目指すのではなく、やりたいことの為にフリーランスの働き方が合ってたと言えるかが重要かと

3ヶ月で12kg痩せた

2016年08月28日 23時14分

最近、会う人会う人に「痩せたねぇ」と言われます。
それもその筈で今年の5月には67kgあった体重が今は54kgまで落とせました。

3ヶ月の体重遷移です。

見事に落ちてますね✨

ダイエットのきっかけは5月に謎の腹痛で内科に行った際、エコー検査で「教科書に乗せられるぐらいの見事な脂肪肝だよw」と先生に言われ脂肪性肝炎と診断されたことです。

実際には帯状疱疹だったのですが、その診断の際に血液検査も行い肝臓の数値が全て悪かったので脂肪性肝炎と診断されてもしょうがなかったのかもしれません。

実際に血液検査で計測した肝臓に関する数値です。

項目 参考基準値 値
r-GTP 70以下 289
AST(GOT) 10-40 89
ALT(GPT) 5-40 166

今までも健康診断で脂肪肝で要経過観察は出ていたんですが「いやぁ、僕フォアグラなんですよ～w」みたいな感じで笑い話ぐらいにしていました。ですが、こうやって誤診の原因になったり「このままでいくと肝臓病に発展するよ」と先生に言われ恐怖を感じダイエットを決意。

脂肪肝は酒をよく飲む人が患うことが多いらしいのですが、最近ではNASH（非アルコール性脂肪性肝炎）という酒をあまり飲まない人の脂肪性肝炎の話もよく出ています。
僕はお酒をあまり飲まないので後者の手前でした。

そして炭水化物大好き人間でした。

全てのおかず（ご飯以外）は米を食べる為のものであり、食事といえば如何に美味しく米を食べれるかという考えです。

うどんもラーメンも焼きそばも米を食べるための味の濃い副菜で、プラス米を食べるので炭水化物の過剰摂取は自覚していました。
ですが好きなものを食べて太るぐらいならまぁいいかと思ってバグバグ食べていました。

ということで太った理由が明らかだったので実行したダイエットは糖質制限です。

まぁあまり神経質にやりすぎてもしんどいので米、パン、麺系は食べないという食生活に変えてみました。

* 主食を豆腐に変えて副菜は肉とか魚、野菜中心
* ジュースは飲まずに水メイン、たまにブラックコーヒーや無糖紅茶
* 21時以降は食べずに0時前ぐらいには寝る

気を付けたのはこれぐらいです。

最初の2～3日は米食べたい、米食べたいと禁断症状に近い症状が出ていたのですがそれを超えると普通に豆腐や野菜で満足感を得れるようになりました。
この峠を越えられるかどうかが糖質制限ダイエットの山場ではないでしょうか。

そしてみるみる減っていく体重が楽しくなってきて、このペースを維持したい為に炭水化物を取るのが嫌に思えてきました。
計測して記録することはとてもモチベーションに繋がるので必ずやるべきです。

先日、目標の57kgを下回ったので血液検査をおこないました。

項目 参考基準値 ダイエット前の値 ダイエット後の値
r-GTP 70以下 289 28
AST(GOT) 10-40 89 25
ALT(GPT) 5-40 166 25

全て正常値に収まっています?

エコー検査でも肝臓に詰まった脂肪は見事に無くなっていました✨

お医者様もこんな短期間に改善した患者は初めてだと驚いていました。先生の「教科書に乗せられるぐらいの見事な脂肪肝だよw」に発奮していたので見返せたような感覚で嬉しかったです。

次はこれを維持する生活習慣と基礎代謝を上げる為に筋肉を増やす努力をしようと思います。

コワーキングスペースを契約してみたけど2ヶ月で解約した話

2016年08月11日 23時27分

今年の2月に猫会社(qnote)を辞め、今ではフリーランスっぽい働き方を初めたのですが、住んでいるマンションで工事が始まり日中はとても煩いので家での作業ができなくなりました。

そこで歩いて行ける距離で電源があるコワーキングスペースや貸し事務所などを探してみたところバッチリなものが見つかりました。

ビズコンフォート
http://bizcomfort.jp/

平日のみの利用で月額5,000円～8,000円ぐらい。金額は場所により変動。
なんとか払ってもいいかなと思える金額だったので早速契約して使ってみました。

設備はホームページで紹介されている通りに綺麗で最初は毎日通って作業していました。

ところが1週間経たない間にそんな好印象は180度変わってしまいます。

問題は利用者です。

不満点一覧↓

  • 食事禁止の張り紙はあるけど気にせずお弁当を食べる
  • シャカシャカと動画の音楽をヘッドホンを使わずに鳴らす
  • 数少ない集中デスク（仕切りのある個人デスク）は何時間も来ない人の場所取り本が常に置かれてほぼ使えない
  • 椅子を複数並べて足を乗せてそり返って本を読む
  • 長い場合は外に出ろ注意書きがあるにも関わらず、いつまでも携帯電話で電話をし続ける

なんで金払って不快な思いをしないといけないのかと怒り心頭に発したので運営に上を伝え張り紙など対応をしてもらったんですが改善なんてするわけもなく、、、
馬鹿らしくなったのでわずか2ヶ月で解約。

初期費用として契約金が1.5万ぐらい取られたのが本当に無駄だったなぁと後悔。

その場所のコワーキングスペースのユーザの常識が極端に無かったのかは分かりませんが、上のようなことが気になる人にはあまり向かないサービスだと思います。

逆に他人が何しようと気にしないという人にとっては、安くて電源も確保できる作業スペースが家や会社以外にも持てる魅力的なサービスです。

結局僕の場合は工事の音が初期よりかなりマシなレベルになったので家で作業するようになりました。

もう少しお金を出してレンタルオフィスにすれば幸せだったのかもしれませんが、月に3～4万あがるくらいなら引越します。

まぁ、こうやって文にすると値段相応って思いになってきました。

頑張って仕事しよう。

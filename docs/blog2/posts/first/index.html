WordpressをSimply Staticで静的サイト化して続きはHugoで書くことにした

2017年8月30日

サーバ移行のタイミングでBlogで使っていたWordpressをSimply Staticで静的サイト化しました。
（ http://www.crossl.net/blog/ ）

よくネックになる速度面はそんなに気になってなかったんですが、2～3ヵ月に1回程度の更新頻度で動的サイトにする必要も感じなかったが主な理由です。

ただSimply Staticでサイトを静的サイト化するのに約50分程度かかったのでこれを記事追加の度にやるのは無理と感じ、続きは別Blogで書くことにしました。

ということでこの記事は静的サイトジェネレータのHugoで書いています。

ぼちぼちテンプレ作りながら更新できるよう頑張ります。

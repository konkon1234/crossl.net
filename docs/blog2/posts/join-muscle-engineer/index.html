「世界初！？筋肉もくもく会」に参加してきた

2019年3月3日

25分もくもく勉強して5分間筋トレするを繰り返す勉強会、「筋肉もくもく会」に参加してきました。

2年前、PowerBlockという高いダンベル買って筋トレやってたんですがフォームが悪かったのか身体を痛めてしまってやめていました。

ウェイトトレーニングはパーソナルトレーナーのような専門家の元でやらないと危ないと思い、調べ始めていた矢先でこんな勉強会を見つけてしまっては参加してしまうでしょう。

今回の発案者の方々、やっぱりマッチョです。

各筋トレは筋肉もくもく会特別トレーナーの星野雄三さんに指導していただきました。

各5分の筋トレ内容

1回目はペアを組んでトレーニング

  • 片手を持ち合ってのスクワット
  • 目をつぶって片足をあげ、相方がちょんちょんしてひたすら耐える
  • 太腿上げ、脹脛上げ

2回目は腕立てを30秒ｘ10秒休憩の4セット

皆さん、腕立ての速度が異常に早い💦

追い込まれるマッスルたち💪🏼#筋肉もくもく会 pic.twitter.com/VGxFiNkL85

— 遠藤@筋肉もくもく会開催３/２💪🏼 (@yukiendooo) 2019年3月2日

3回目はバービージャンプ

1回20秒やって10秒休憩の4セット、余裕のある人は+2セットを限界がくるまで…

僕は6セットで脱落でした、脱落時は「負け犬です」と宣言して着席するという負けたくないという気持ちを刺激するルールで自分の限界を突破させるという軍隊式。

優勝者は13回だったかな、マッチョな指揮官に表彰されていましたw

4回目はストレッチ

参加してみた感想

端的に言うととても楽しかったです。

最初の筋トレをペアトレにすることで周りの人と会話をする機会を作り、会場の雰囲気を和らげ良い雰囲気を作り上げる辺りにとても巧な気配りを感じました。

会場に着いた瞬間はスタッフの方々がマッチョすぎて入っていいものか戸惑いましたがｗ

各筋トレの合間には25分間のもくもくタイムがあり、僕はそこではNSCAジャパン発行のストレングス＆コンディショニングという本で筋肉について勉強してきました。

理論と実践という形で身になりましたが、筋トレの強度が僕には強すぎて後半はほぼ休憩タイムになってしまったのが反省点です。

目的だった筋肉に関する質問はタイミングを掴めず伺うことができなかったのが心残りですが、星野さんがやっているBUDDY TRANINGのチケットをいただけたので機会を作って行ってみます。

素晴らしい機会を作っていただいたスタッフの皆様、会場を貸していただいた渋谷100BANCH様、ありがとうございました。
